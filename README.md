# Azure-For-Beginner.github.io

## 초보자를 위한 Azure Documents
* 당신도 이것만 따라하면 Azure 고수가 될 수 있다?!

## 제작 이유
* 이제는 옛날 자료가 되어버린 Microsoft Documents
* Microsoft Documents에는 없는 오류들이 존재한다
* Google 검색을 해도 제대로된 결과를 얻을 수 없다
